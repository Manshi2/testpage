Chat Widget — iOS Focus Isolation Test

Open this page on iOS Chrome. Start with everything OFF, open the chat, and confirm the input works. Then flip scripts ON one at a time (reload after each toggle) and retest until the bug reappears.

How to use this on your phone
  1. Load this page on the iOS device where the bug reproduces.
  2. Leave all toggles OFF, tap "Open Chat", tap the input field. It should focus normally.
  3. Turn ON AudioEye only, reload the page, retest the input.
  4. If it still works, turn AudioEye OFF, turn ON Hotjar only, reload, retest.
  5. Repeat down the list. The one that reproduces the bug is your root cause.
  6. Check the black log box below — it timestamps focus/blur events on the widget's input so you can see exactly how fast the blur happens after focus.
